# 🌍 AirQualityAI – Global AQI Prediction Web App with Interactive Map

**Real-time & forecasted Air Quality Index (AQI) prediction for the entire world** — featuring an interactive global map, smart city search, AI-powered chatbot, personalized health recommendations, historical trends, and country-wise comparisons.

Stay informed about air pollution anywhere on Earth. Whether you're planning travel, monitoring your city, protecting your health, or researching global air quality trends, this open-source web app delivers accurate insights and actionable advice.

### ✨ Key Features
- **Worldwide AQI Prediction** – Real-time and future forecasts for any location globally
- **Interactive World Map** – Color-coded AQI visualization with zoom and layer support
- **Smart Search** – Instantly find any city, country, or coordinates
- **AI Chatbot** – Ask natural questions about air quality and get instant intelligent answers
- **Personalized Recommendations** – Health, activity & lifestyle tips based on current AQI and your profile
- **AQI History & Trends** – Beautiful interactive charts showing past data and patterns
- **Global AQI Comparison** – Compare countries and cities with live leaderboards
- **Complete User Guide** – Step-by-step instructions on how to use every feature

Built for travelers, health-conscious users, researchers, and environmental enthusiasts who need reliable **air quality monitoring** and **AQI forecasting**.

### 🚀 Why Use AirQualityAI?
- Accurate global AQI predictions powered by advanced models
- User-friendly interface with responsive design
- Focus on actionable insights to help you breathe cleaner air

## Run Locally
**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
